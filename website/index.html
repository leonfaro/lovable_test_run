How it works

No bundler, no build step. The landing page is static HTML + CSS. The MVP demo is a single React file loaded from CDN modules and mounted into the demo container.

1
Backend APIs

FastAPI serves dashboard data and the static files at /index.html and /App.jsx.

2
Landing page

This page lives in website/index.html with a dedicated #demo-root placeholder.

3
React mount

App.jsx detects #demo-root first, then falls back to #root for standalone use.

Live demo

Run the backend, open this page, and use the embedded cockpit to walk through coverage, market share and scenario shifts. All data is synthetic.

How to explore

  1. Click a Leistungsgruppe in the matrix to focus the charts.
  2. Choose Status quo, Ausbau or Rueckzug, then move the slider.
  3. Watch Deckungsgrad, Marktanteil and CHF impact update live.
  4. Use the table search to filter by category or group name.

Tip: start backend via make run or uvicorn backend.main:app --reload --port 8000.
